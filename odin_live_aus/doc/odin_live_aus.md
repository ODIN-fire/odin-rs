# odin_live_aus

This is an example application for a ODIN web server showing Australian data sources. It currently covers
the following public data sources (no build features yet):

- basic [`odin_cesium`](../odin_cesium/odin_cesium.md) services (basic clock, view and imagery)
- [`odin_share`](../odin_share/odin_share.md) service (both for shared data and messaging)
- [`odin_geolayer`](../odin_geolayer/odin_geolayer.md) to show configured GeoJSON data sets
- [`odin_goesr](../odin_goesr/odin_goesr.md) geostationary satellite hotspots
- [`odin_orbital](../odin_orbital/odin_orbital.md) various orbiting satellites
- [`odin_wind`](../odin_wind/odin_wind.md) micro grid wind simulation and display with WindNinja (using Open-Meteo data)
- [`odin_bushfire`](../odin_bushfire/odin_bushfire.md) near realtime bushfire perimeters from [Geoscience Australia](https://ecat.ga.gov.au/geonetwork/srv/eng/catalog.search#/home)
